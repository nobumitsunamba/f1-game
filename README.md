# F1 2026 鈴鹿サーキット レーシングシミュレーター

ブラウザで動く 3D F1 レーシングシミュレーターです。2026年シーズンのレギュラードライバー22名から1人を選び、実データに基づく鈴鹿サーキットをタイムアタックで走ります。

![Three.js](https://img.shields.io/badge/Three.js-r184-blue) ![Vite](https://img.shields.io/badge/Vite-6-purple)

## 起動方法

```bash
npm install
npm run dev      # http://localhost:5173 を開く
```

本番ビルドは `npm run build` → `dist/` を任意の静的ホスティングへ。

## 特徴

### 鈴鹿サーキット(実データ)
- センターライン座標と路面幅は実測データ([TUMFTM/racetrack-database](https://github.com/TUMFTM/racetrack-database))を使用。全長 5.80km、S字・逆バンク・デグナー・ヘアピン・スプーン・130R・カシオトライアングルまで実コース通りのレイアウト
- F1で唯一の**立体交差(8の字)**を再現 — デグナー2を抜けるとバックストレートの橋の下をくぐります
- 実コースに準じた約40mの高低差(1コーナーへの下り、S字〜ダンロップの登り等)
- 縁石、グランドスタンド、ピットビル、シグナルゲート、コーナー看板、ブレーキングマーカー、観覧車

### 2026年マシンと物理エンジン
- 2026年レギュレーション準拠のパラメータ: 車重768kg+燃料、ホイールベース3.4m、8速ギア
- **アクティブエアロ(Xモード/Zモード)**: ストレートで自動的に(またはXキーで手動)低ドラグモードに切替
- **ERSオーバーテイクブースト**(Spaceキー、周回毎のエネルギー収支とブレーキ回生あり)
- タイヤモデル: スリップ角ベースの横力カーブ、フリクションサークル(縦力と横力の食い合い)、荷重感度、空力ダウンフォースと前後荷重移動
- 路面: アスファルト/縁石/芝・グラベルでグリップと抵抗が変化
- 240Hz固定ステップ積分

### ゲーム機能
- 11チーム22名の2026年ドライバー選択(チームカラーのマシンで走行)
- シグナル5灯消灯でスタート
- ラップ/セクタータイム計測、ベストラップとのリアルタイムデルタ表示
- **ゴーストカー**(自己ベストラップを半透明マシンが再走、localStorageに保存)
- カメラ4種: チェイス / コックピット(HALO付き) / TV中継 / ドライバー視点(車体非表示)
- エンジン音(WebAudio合成、回転数連動)、ミニマップ、コーナー名コールアウト
- ゲームパッド対応(左スティック+トリガー)

## 操作

| キー | 機能 |
|---|---|
| ↑ / W | アクセル |
| ↓ / S | ブレーキ |
| ← → / A D | ステアリング |
| Space | ERSオーバーテイクブースト |
| X | アクティブエアロ(Xモード)手動展開 |
| E / Q(Shift / Ctrl) | シフトアップ / ダウン |
| C | カメラ切替 |
| R | リスポーン |
| M | ミュート |
| Esc | メニューに戻る |
| F1〜F5 | アシスト切替(TC / ABS / スタビリティ / オートシフト / 自動エアロ) |

## 2026年グリッド

マクラーレン(ノリス/ピアストリ)、フェラーリ(ルクレール/ハミルトン)、レッドブル(フェルスタッペン/ハジャー)、メルセデス(ラッセル/アントネッリ)、アストンマーティン(アロンソ/ストロール)、アルピーヌ(ガスリー/コラピント)、ウィリアムズ(アルボン/サインツ)、レーシングブルズ(ローソン/リンドブラッド)、アウディ(ヒュルケンベルグ/ボルトレート)、ハース(オコン/ベアマン)、**キャデラック(ペレス/ボッタス)** の11チーム22名。

※ レースナンバーは原則として各ドライバーのパーマネントナンバーを使用しています。一部の新人のナンバーは暫定です。

## 開発者向け

```bash
npm run validate-track      # コース形状の検証(全長・橋のクリアランス・ASCIIマップ)
node tools/e2e-smoke.cjs    # ヘッドレスE2E(要: npm i --no-save puppeteer + dev server)
node tools/convert-track.js # data/suzuka-centerline.csv から座標モジュールを再生成
```

ブラウザコンソールの `window.__sim` でテレポート(`__sim.teleport(s)`)、オートパイロット(`__sim.autopilot = true`)、物理の早送り(`__sim.simulate(秒)`)が使えます。

## データ出典 / ライセンス

- コース座標データ: [TUMFTM/racetrack-database](https://github.com/TUMFTM/racetrack-database)(LGPL-3.0)。センターラインはOpenStreetMapのGPS点列を平滑化したもの、コース幅は衛星画像から抽出されたものです。`data/suzuka-centerline.csv` として同梱し、`src/track/suzukaData.js` に変換しています。高低差プロファイルは実コースの公表値を参考にした本プロジェクト独自のモデルです。
- コード: MIT
